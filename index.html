Low-Code Data Distributed Processing Framework
Repository

A configuration-driven framework that eliminates boilerplate in data pipeline development. Define workflows as data — the system generates and orchestrates them automatically.

Apache Airflow Apache Spark Apache Iceberg MinIO Docker Oracle PDB PostgreSQL
Dynamic DAG Generation — Airflow DAGs generated at runtime from database configuration — no Python code per workflow
Control Table Architecture — 6 relational tables (streams, process groups, processes, dependencies, schedules, logging) as single source of truth
Data Lakehouse Pattern — Spark on Iceberg open table format over MinIO — ACID transactions and schema evolution on object storage
Multi-tier Data Flow — Raw staging → Dimension/Fact tables → Aggregations, with Oracle PDB as downstream warehouse
Containerized Stack — 5 services orchestrated via Docker Compose for reproducible local and production environments

cc-mimic — AI Coding Agent from Scratch
Repository

A terminal-based AI coding agent built from scratch — featuring tool execution, streaming responses, and session management. No agent frameworks, just clean async Python.

Python 3.12+ asyncio Rich Terminal Click CLI MCP
Agent Loop Architecture — Core reasoning-action-observation cycle: prompt → tool selection → execution → context feedback
Tool Registry — Extensible system with 4 approval levels (auto-ask, allowlist, deny, full-auto) for controlled execution
Streaming Responses — Real-time token streaming via Rich terminal UI for responsive feedback on long-running tasks
Session Persistence — Conversations checkpointed and resumable with full message history preserved across sessions
MCP Integration — Model Context Protocol support for connecting external tool servers — open standard for agent-tool communication

Google Cloud Vertex AI Google ADK Terraform GitHub Actions OpenTelemetry
